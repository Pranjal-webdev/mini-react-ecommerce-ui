# mini-react-ecommerce-ui
Responsive e-commerce frontend application built with react featuring product filtering,searching,sorting,cart functionality , dark/light theme and mobile responsive design.
---
## Features
- Search Products
- Category Filtering
- Price Filtering
- Product Sorting
- Add To Cart Functionality
- Remove From Cart
- Dark / Light Theme Toggle
- Responsive Design
- No Products Found UI
- Dynamic Product Rendering

  ---
## Technologies used
- React.js
- Javascript
- Bootstrap
- CSS3
- React Hooks
